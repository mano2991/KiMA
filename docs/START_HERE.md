# START HERE

KiMA has been cleaned up into a simpler folder structure, but the app still runs the same way.

## Fast Start

```bash
npm install
npm start
```

Open `http://localhost:8282`.

## Main Folders

```text
src/         main application source
samples/csv/ example datasets
legacy/      archived standalone HTML file
docs/        supporting notes from version one
```

## Main Commands

- `npm start` runs the webpack dev server on port `8282`.
- `npm run build` creates a production build in `dist/`.
- `npm run serve` serves the built `dist/` folder on port `8282`.
