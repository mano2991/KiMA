About KiMA

KiMA is built to analyze locomotor data from different experiment types without mixing the logic between them.

Each analysis block has its own parser, settings, plots, videos, and exports so the modules stay separate and stable.

The tools page is where users open the actual analysis modules. This home page is for explaining the tool and guiding new users.

KiMA can also point users back to the Venkatesh Lab website for the lab overview and related work.

Required Labeling

Ladder / Treadmill: use `toe`, `mtp`, `ankle`, `knee`, `hip`, and `iliac crest`.

Foot Fall / CatWalk: use `RF`, `LF`, `RH`, and `LH` with `x`, `y`, and `likelihood`.

PCA: use the aggregated metrics CSV exported from KiMA multiple-sample analysis.

If the labels are different, the module will not parse correctly, so keeping the labeling proper is important.

Modules In KiMA

Ladder & Rung: single run visualization and gait parameters.

Foot Fall: paw contact detection, paw plots, and paw videos.

CatWalk: stride length and stride width from paw placement.

Treadmill: cycle-based kinematics and treadmill playback.

Multiple Sample and PCA: group-level analysis and separation.

Stride width is computed as the lateral separation between left and right fore or hind paw contacts.
